## 카테고리별 이슈 분류

### A. 공통 이슈 — PushNotificationManager (전체 페이지)
**영향도**: 🔴 전체 27페이지
**원인**: `components/etc/PushNotificationManager.tsx` 내 한국어 하드코딩

```
"알림 받기"
"새로운 쿠폰 정보를 푸시 알림으로 받아보세요!"
"나중에"
```

**수정**: `messages/en.json`의 `pushNotification` 네임스페이스 추가 + `useTranslations()` 적용

---

### B. 게임 데이터 한국어 이름 (영어 이름 누락)
**영향도**: 🔴 캐릭터 목록, 스킨 목록, 공명/의지 설정, 추천 조합, 광상 가이드, 의지 가이드
**원인**: 일부 캐릭터/아이템 데이터에 `engDisplayName` 또는 `engName` 필드 없음

확인된 한국어 이름 예시:
- 캐릭터: 정화광, 노사전, 이고르, 마샤, 브룸, 베릴, 코르부스, 루부스카, 카론, 센티널, 곡랑, 머큐리아, 카카니아, 빌라, 산귀
- 스킨: 고독의 찬송가, 맟춤형 봄 의상, 미치광이 도인, 속세를 계산하며, 그림자를 따르는 노래 등
- 배너: "[3.3] 마샤 픽업", "[3.3] 이고르 픽업"
- 의지 이름: 천체의 흉조, 연극의 굴절, 뼈의 늑대, 원시적 기쁨, 새 시대 선언자

**수정**: 데이터 파일(`data/characters/`, `data/banners.ts`)에 영어 이름 추가

---

### C. UI 하드코딩 텍스트

#### C-1. 뉴비 가이드 FAQ (`components/newbie_guide/NewbieGuide.tsx`)
**영향도**: 🔴 87개
FAQ 전체가 한국어 하드코딩:

```
"무과금이면 얼마나 뽑을 수 있나요?"
```

**수정**: `newbieGuide.faq` 번역 키 추가 또는 FAQ 데이터를 `data/newbie_guide.ts`로 분리 후 locale별 처리

#### C-2. 추천 조합 역할/설명 텍스트 (`components/recommend_team/RecommendTeam.tsx`)

**수정**: `recommendTeam` 네임스페이스에 역할명 + 태그 번역 키 추가

#### C-3. 빙고 카드 내용 (`components/bingo/Bingo.tsx`)

**수정**: 빙고 데이터를 locale별로 분리하거나 영어 번역 추가

#### C-4. 캐릭터 퀴즈 모달 aria-label

**수정**: `common.confirmModal` 키로 교체

#### C-5. 미래시 이벤트 이름
`원정기` 등 일부 이벤트 종류 이름
**수정**: `futureInsight` 네임스페이스에 이벤트 타입 번역 키 추가

---

### D. 데이터 연동 이슈 (getLocalizedName 미적용)

캐릭터 상세 페이지(`/en/character/charlotta`)는 ✅ 클린이지만,
캐릭터 목록(`/en/character`)은 🔴 226개 한국어 발견.

→ 캐릭터 이름을 렌더링하는 **목록 컴포넌트**에서 `getLocalizedName()` 미적용 또는
→ 해당 캐릭터들의 데이터에 `engDisplayName` 필드 자체가 없음
